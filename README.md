# 🚀 Portfólio de Projetos Arduino

Este é um portfólio web moderno e responsivo desenvolvido para exibir uma coleção de projetos e códigos para Arduino. O objetivo é documentar e compartilhar soluções que vão desde conceitos básicos até lógica complexa de automação e reprodução de melodias.

## 🛠️ Tecnologias Utilizadas

- **Web Front-end**:
    - HTML5 Semântico
    - CSS3 (Vanilla com Variáveis e Design Responsivo)
    - JavaScript (ES6+ para renderização dinâmica)
- **Hardware/Embedded**:
    - Linguagem C++ (Arduino)
    - Plataforma Arduino (Uno/Nano/Mega)

## 📂 Projetos em Destaque

O portfólio apresenta diversos códigos prontos para uso, incluindo:

- 🎵 **Melodias com Buzzer**: Super Mario Theme, Marcha Imperial (Star Wars) e Game of Thrones, todos sincronizados com efeitos de LEDs.
- 🚥 **Sistemas de Controle**: Lógica de semáforo e sequenciadores usando laços `for`.
- 🤖 **Automação (APAC)**: Integração de sensores e relés para sistemas autônomos.
- 💡 **Fundamentos**: O clássico Blink (Piscante) e manipulação de portas digitais/analógicas.

## 🚀 Como Visualizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/robertoatila/portfolio-arduino.git
   ```
2. Abra o arquivo `index.html` no seu navegador.
